# 🚀 クイックスタートガイド

VRChat ログビューアーを5分で使い始めましょう！

---

## ⚡ 3ステップで起動

### ステップ1: ファイルをダウンロード

以下の5つのファイルを同じフォルダにダウンロード:

- ✅ `vrchat_log_viewer_modular.py`
- ✅ `constants.py`
- ✅ `models.py`
- ✅ `utils.py`
- ✅ `ui_builder.py`

### ステップ2: Pythonをインストール（未インストールの場合）

**Windowsの場合:**
1. https://www.python.org/ にアクセス
2. 「Download Python」をクリック
3. インストーラーを実行
4. ⚠️ **重要**: 「Add Python to PATH」にチェック！

**確認方法:**
```bash
python --version
```
Python 3.7以上ならOK！

### ステップ3: アプリを起動

**方法1: ダブルクリック（簡単）**
- Windows: `start.bat` をダブルクリック
- Mac/Linux: `start.sh` をダブルクリック

**方法2: コマンドライン**
```bash
python vrchat_log_viewer_modular.py
```

## 📖 初めて使う方へ

### 起動したら...

1. **ログパスを確認**
   - 自動的にVRChatログフォルダが検出されます
   - 通常: `C:\Users\<ユーザー名>\AppData\LocalLow\VRChat\VRChat`

2. **ログファイルを選択**
   - 上部のコンボボックスから見たいログを選択
   - 最新のログが一番上に表示されます

3. **フィルターを使ってみる**
   - 検索ボックスにキーワードを入力
   - ログレベル（Debug/Info/Warning/Error）をチェック

4. **グループメッセージを見る**
   - 右パネルで「グループ」を選択
   - 通知メッセージが自動分類されています

## 💡 よく使う操作

### ログを検索
```
1. Ctrl+F を押す
2. キーワードを入力
3. 自動でフィルタリング！
```

### ログをコピー
```
1. ログを選択（複数選択可）
2. Ctrl+C でコピー
3. メモ帳などに貼り付け
```

### グループメッセージをエクスポート
```
1. 見たいグループを選択
2. 「エクスポート」ボタンをクリック
3. ファイル形式を選択（.txt または .json）
4. 保存先を指定
```

## ❓ よくある質問

### Q: ログが表示されない
**A:** ログフォルダのパスを確認
```
メニュー → ファイル → ログフォルダを選択
→ VRChatのログフォルダを指定
```

### Q: エラーが出る
**A:** Pythonのバージョンを確認
```bash
python --version
# Python 3.7以上が必要
```

### Q: 動作が重い
**A:** フィルターを活用
```
- 不要なログレベルのチェックを外す
- 検索でログを絞り込む
- 古いログファイルは別フォルダへ
```

## 🎨 カスタマイズしたい方へ

### ウィンドウサイズを変更
`constants.py` を開いて:
```python
WINDOW_GEOMETRY = "1400x800"  # お好みのサイズに
```

### 色を変更
`constants.py` を開いて:
```python
LOG_COLORS = {
    'error': '#f48771',  # エラーの色
    'info': '#4fc1ff',   # 情報の色
    ...
}
```

詳しくは `README.md` の「カスタマイズ」セクションを参照！
