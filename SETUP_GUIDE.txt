# VRChat ログビューアー v2.1 - セットアップガイド 📚

## 🎯 ファイル構成の説明

```
VRChatLogViewer_v2.1/
│
├── vrchat_log_viewer.pyw  ← 【これを起動】メインの起動ファイル
├── debug_start.py         ← トラブル時のデバッグ起動用
├── start_with_error_log.pyw ← エラーログ付き起動
├── README.txt             ← 使い方ガイド
│
└── lib/                   ← 【重要】すべてのモジュール
    ├── vrchat_log_viewer.py   ← メインアプリケーション
    ├── virtual_treeview.py    ← 仮想レンダリング（高速化）
    ├── async_loader.py        ← 非同期ローダー
    ├── constants.py           ← 定数定義
    ├── models.py              ← データモデル
    ├── progress_dialog.py     ← 進捗ダイアログ
    ├── utils.py               ← ユーティリティ
    ├── ui_builder.py          ← UI構築
    ├── theme_manager.py       ← テーマ管理
    ├── plugin_manager.py      ← プラグインシステム
    └── README.md              ← モジュール詳細説明
```

---

## 📥 インストール手順

### 1. ファイルの配置

**すべてのファイルを同じフォルダに展開してください**

重要なポイント:
- `vrchat_log_viewer.pyw` と `lib/` フォルダが **同じ階層** にあること
- `lib/` フォルダの中身を移動したり削除したりしないこと

### 2. 動作確認

1. `vrchat_log_viewer.pyw` をダブルクリック
2. エラーが出なければ成功！

---

## ⚠️ トラブルシューティング

### "lib/フォルダが見つかりません" エラー

**原因**: ファイル構成が正しくありません

**解決方法**:
1. すべてのファイルを同じフォルダに配置
2. 以下のような構成になっているか確認:
   ```
   あなたのフォルダ/
   ├── vrchat_log_viewer.pyw
   ├── debug_start.py
   └── lib/
       ├── vrchat_log_viewer.py
       ├── virtual_treeview.py
       └── (他のファイル)
   ```

### "モジュールが見つかりません" エラー

**原因**: `lib/` フォルダ内のファイルが不足しています

**解決方法**:
1. `lib/` フォルダに以下のファイルがあるか確認:
   - vrchat_log_viewer.py
   - virtual_treeview.py
   - async_loader.py
   - constants.py
   - models.py
   - progress_dialog.py
   - utils.py
   - ui_builder.py
   - theme_manager.py
   - plugin_manager.py

2. 不足している場合は、再ダウンロードしてください

### 起動はするがエラーが出る

**解決方法**:
1. `debug_start.py` を実行
2. 表示されたエラーメッセージを確認
3. エラー内容をコピーして開発者に報告

## 🎨 カスタマイズ

### テーマの変更
1. アプリを起動
2. メニュー → 表示 → テーマを選択
3. 5種類のプリセットから選択可能

### プラグインの追加
1. メニュー → ツール → プラグイン管理
2. プラグインファイル(.py)を選択して追加
