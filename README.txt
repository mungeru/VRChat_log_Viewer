## ⚠️ トラブル時

### 起動しない
→ **`debug_start.py`** を実行してエラーを確認

### "lib/フォルダが見つかりません" エラー
→ すべてのファイルを正しく配置してください
→ **lib/** フォルダがメインの **.pyw** ファイルと同じ場所にあるか確認

### ログが表示されない
→ メニュー → ファイル → ログフォルダを選択

### 動作が重い
→ フィルターで行数を減らす
