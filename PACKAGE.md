## 🚀 インストール手順

### ステップ1: ファイルをダウンロード

**必須ファイル（5個）** を同じフォルダにダウンロード:
- ✅ vrchat_log_viewer_modular.py
- ✅ constants.py
- ✅ models.py

- ✅ ui_builder.py

**推奨**: 起動スクリプトもダウンロード

Python 3.7以上が必要です:

- Mac/Linux: `start.sh` をダブルクリック

```bash
python vrchat_log_viewer_modular.py
```

## ✨ 主な機能

- 🔍 **高速検索**: キーワードでログを瞬時に絞り込み
- 📊 **スマートフィルタ**: ログレベル別に表示切替
- 📨 **グループメッセージ**: 通知を自動分類
- 🔄 **リアルタイム更新**: VRChat起動中も自動更新
- 📋 **コピー機能**: ログを安全にコピー（読み取り専用）
- ⌨️ **ショートカット**: Ctrl+F, Ctrl+C など
- 🎨 **ダークテーマ**: 目に優しいデザイン
- ⚡ **高速表示**: 10万行でもスムーズ

## ⚠️ 注意事項

### 必ずお読みください

1. **ファイルの配置**
   - 5つのPythonファイルは **必ず同じフォルダ** に配置
   - ファイル名を変更しないでください

2. **Python環境**
   - Python 3.7以上が必要
   - tkinterが含まれていること（通常は標準搭載）

3. **ログファイルへのアクセス**
   - VRChatログフォルダへの読み取り権限が必要
   - デフォルト: `C:\Users\<ユーザー名>\AppData\LocalLow\VRChat\VRChat`

4. **設定ファイル**
   - `vrchat_group_names.json` は自動生成されます
   - このファイルにグループ名の設定が保存されます

## 🐛 トラブルシューティング

### Q: 起動しない
**A**: Pythonのバージョンを確認
```bash
python --version  # 3.7以上か確認
```

### Q: "モジュールが見つかりません"エラー
**A**: すべてのPythonファイルが同じフォルダにあるか確認

### Q: ログが表示されない
**A**: ログフォルダのパスを確認
- メニュー → ファイル → ログフォルダを選択

詳しくは **README.md** の「トラブルシューティング」を参照
